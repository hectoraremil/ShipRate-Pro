# Base de datos

Scripts iniciales de SQL Server para crear el esquema y cargar las tarifas base.

Orden recomendado:

1. Ejecutar `database/001_create_schema.sql`.
2. Ejecutar `database/002_seed_data.sql`.

Datos iniciales:

- `India` = `5.00 USD` por kilogramo
- `Estados Unidos` = `8.00 USD` por kilogramo
- `Reino Unido` = `10.00 USD` por kilogramo
